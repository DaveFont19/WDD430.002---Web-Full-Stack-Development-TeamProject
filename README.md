# WDD430.002---Web-Full-Stack-Development-TeamProject

## Installation Instructions

 * Clone this repository
 * Enter the directory `wdd430-team-project`
 * Run `npm install .`
 * Test your installation by running `npm run dev`

User Stories:

 1. As a customer, rate products 
 1. As a seller, you need to create an authentication profile 
 1. As seller, add products 
 1. Customer, Create a shopping cart 
 1. As a customer, search for products or sellers 
 1. Seller, Product details 
 1. As a seller, CRUD products 
 1. As a customer, check the products in the cart, and add payment details 
 1. Filter category products 
 1. As a customer, save a product or a wishlist
